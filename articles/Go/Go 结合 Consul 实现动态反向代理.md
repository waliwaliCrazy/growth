# 实现逻辑

一共分成几个步骤：

1. 代理接收到客户端的请求，复制了原来的请求对象
2. 根据一些规则，修改新请求的请求指向
3. 把新请求发送到根据服务器端，并接收到服务器端返回的响应
4. 将上一步的响应根据需求处理一下，然后返回给客户端
